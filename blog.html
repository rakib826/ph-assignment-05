How does JavaScript access the DOM?

JavaScript accesses the DOM using methods like getElementById, querySelector, and getElementsByClassName.

How do you modify text content in the DOM?

Use element.textContent or element.innerHTML to change the text or HTML of an element.

What is event delegation in the DOM?

Event delegation allows events to be handled at a higher level, reducing the need for multiple event listeners.

How can you remove an element from the DOM?

Use element.remove() or parentNode.removeChild(childNode) to remove an element.
